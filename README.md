The Camera Chromaticity & Gamut Mapping Tool is a self-contained, single-file HTML application that visualises the colour gamuts of professional digital cameras and compares them against industry-standard target colour spaces. It requires no server, no installation, and no network connection — open the HTML file in any modern browser.
The tool is built on the CIE 1931 xy chromaticity diagram and presents two complementary plot types: a traditional CIE 1931 xy diagram and an xy polar plot centred on the working white point. Camera characteristics are derived from DNG ForwardMatrix 2 values extracted with ExifTool. A seperate X2DII camera matrix - X2DII CM1 was added and derived directly from the embedded color matrix of the .3fr file.

The Colourspace Gamut Mapping Tool is a self-contained, single-file HTML application that visualises industry-standard colour spaces on the CIE 1931 xy chromaticity diagram and polar plot, and applies five distinct gamut mapping strategies to show how colours are transformed when converting from one colour space to another.

The Luminance-Aware Gamut Mapping Tool (v3) is a companion to the xy chromaticity gamut mapping tool that adds the luminance dimension missing from pure chromaticity analysis. For any source→target conversion across 17 colour spaces spanning D50, D65, and D60 native white points, it derives the full RGB→XYZ matrix from each space's primary chromaticities and white point, extracting the Y luminance coefficients that quantify each channel's contribution to overall brightness — for sRGB these are the canonical 0.2126 / 0.7152 / 0.0722. Primaries and gamut boundaries are converted to CIELAB and rendered in four views selectable via tooltipped buttons: the a*b* plane (hue and chroma), the L*C* plane (lightness vs saturation, C* axis extended to 360 to accommodate ProPhoto's extreme primaries), L*h° polar (hue angle with L* as radius), and a fully interactive 3D LCh view showing the gamut bodies as colour-spectrum-filled solids in cylindrical Lab space. The 3D view renders three independently toggleable elements — the source gamut as a solid-line wireframe cage, the target as a dashed wireframe cage, and the mapped gamut as a depth-sorted colour-filled solid with Lambertian shading and Lab→sRGB face colouring — all rotatable by click-drag and scalable via a 30–200% zoom slider defaulting to 50%. Gamut boundaries throughout are sampled at 180 angular intervals with luminance computed via RGB→XYZ edge parametrisation, and each strategy produces a geometrically distinct Lab representation: Compress shows a uniformly scaled source boundary; Relative Colorimetric shows the WP-translated and clipped boundary; Hard Clip and Hue-Preserve show the source∩target intersection; and Chroma Map renders a 50% chroma contour illustrating its interior scaling. The analysis panel reports Y-coefficients, per-primary CIELAB values with ΔL* impact ratings, and a narrative covering luminance weighting, per-channel lightness implications, mapping outcomes, and a grey balance verdict — all exportable as a structured plain-text report.
